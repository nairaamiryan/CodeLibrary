## 🛠️ Օգտագործված տեխնոլոգիաներ

- **HTML5** — կառուցվածք
- **CSS3** — ոճ, անիմացիաներ, responsive դիզայն
- **Vanilla JavaScript** — FAQ accordion ֆունկցիոնալություն
